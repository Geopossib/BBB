# **App Name**: BSN Connect

## Core Features:

- Admin Content Upload: Secure admin interface for uploading videos, articles, and audio content.
- Article Display: Display blog articles with integrated images for users to view.
- Video Streaming: Enable users to watch videos uploaded by the admin, similar to a YouTube feature.
- Audio Playback and Download: Allow users to play audio and download audio files uploaded by the admin.
- Content Categorization and Search: Categorize content (videos, articles, audio) and implement a search function to enhance content discovery.
- User Authentication: Basic authentication mechanism, including password management.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to represent trust and stability.
- Background color: Very light blue (#F0F4FF), almost white, creating a clean and calming atmosphere.
- Accent color: Golden yellow (#FFC107) for highlights and call-to-action buttons, symbolizing enlightenment and positivity.
- Headline font: 'Playfair', serif, for an elegant, high-end feel. Body Font: 'PT Sans', sans-serif. 
- Use clean and modern icons that reflect the themes of faith and community.
- A clear, intuitive layout to enhance user experience. Prominent placement of video and audio content.
- Subtle transitions and animations for a smooth and engaging user experience.
